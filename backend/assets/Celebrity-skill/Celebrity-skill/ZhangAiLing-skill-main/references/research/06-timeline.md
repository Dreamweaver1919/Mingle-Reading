# 张爱玲生平时间线（1920-1995）

**调研时间**：2026年5月18日  
**来源**：传记、年谱、书信集、自传性散文

---

## 一、童年与家族（1920-1938）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1920.09.30 | 生于上海麦根路（今泰兴路），本名张煐 | 出生于没落贵族家庭，祖父张佩纶（清末大臣），外曾祖父李鸿章 |
| 1922 | 随家迁居天津 | 父亲张志沂在天津谋职 |
| 1924 | 母亲黄逸梵与父亲不和，赴英国留学 | 现代型母亲 vs 遗少型父亲，构成她性格的双重矛盾来源 |
| 1928 | 随家迁回上海 | 开始在英文小学就读 |
| 1930 | 父母正式离婚，母亲再赴欧洲 | 改名"张爱玲"，与弟弟留在父亲处 |
| 1931 | 入读上海黄氏小学 | 开始接触西方文学和英语教育 |
| 1937 | 继母何干丽进入家门，家庭关系恶化 | 父亲因继母言语打了张爱玲，后将其关禁闭数月 |
| 1938 | 逃出父家，投奔母亲 | 彻底与父亲决裂，开始独立生活准备 |

---

## 二、求学与成名前期（1939-1942）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1939 | 发表《天才梦》，获《西风》月刊征文奖 | 首次公开发表，"生命是一袭华美的袍，爬满了蚤子"初现 |
| 1939 | 赴香港大学就读，主修文科 | 离开上海，英文文学教育的系统训练 |
| 1941 | 港大成绩优异，期末考第一 | 证明其卓越的语言能力 |
| 1941.12 | 太平洋战争爆发，日军轰炸香港 | 亲历战火，后写入《烬余录》 |
| 1942.01 | 香港沦陷，港大停办，被迫回上海 | 未能获得英国文凭，成为她一生的遗憾之一 |
| 1942 | 回到沦陷的上海，依靠母亲、自谋生路 | 开始真正以写作谋生 |

---

## 三、创作黄金期与成名（1943-1945）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1943.05 | 《沉香屑——第一炉香》发表于《紫罗兰》 | 首篇重要小说，一举引发关注 |
| 1943.07 | 《倾城之恋》发表于《杂志》 | 确立她在上海文坛的地位 |
| 1943.10 | 《金锁记》发表于《杂志》 | 夏志清后称"中国从古以来最伟大的中篇小说" |
| 1943 | 结识胡兰成（汪伪政权宣传部官员） | 改变她一生的情感关系开始 |
| 1944.02 | 散文集《流言》出版 | 确立散文风格，"参差的对照"美学宣言 |
| 1944.06 | 小说集《传奇》出版，一天之内售罄 | 上海文坛轰动，达到创作影响力顶峰 |
| 1944.08 | 与胡兰成以私婚形式结合 | 婚书："愿使岁月静好，现世安稳" |
| 1945.01 | 傅雷（迅雨）发表《论张爱玲的小说》 | 首篇重要批评，批评《连环套》，张爱玲以《自己的文章》回应 |
| 1945.08 | 日本投降，抗战胜利 | 胡兰成成为逃亡的汉奸，张爱玲的处境开始恶化 |

---

## 四、情感破裂与离开上海（1946-1952）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1946 | 赴温州探望流亡中的胡兰成 | 目睹胡兰成与护士范秀美同居，沉默离开 |
| 1947 | 写信给胡兰成，提出离婚，附上30两黄金 | 以物质清算完成情感清算，"你已不是我认识的人了" |
| 1948 | 创作相对低产，上海文学生态改变 | 左翼文学逐渐成为主流，她的写作与时代脱节 |
| 1949.10 | 中华人民共和国成立 | 张爱玲留在上海，尝试适应新文学秩序 |
| 1950 | 参加上海第一次文代会 | 尝试融入新秩序，但其写作方向与时代要求完全相悖 |
| 1950-51 | 去农村参加土改体验生活 | 为《秧歌》积累素材，但内心已决定不能留在大陆 |
| 1952.07 | 以"回港大完成学业"为由申请离境 | 离开上海，永远没有回来 |

---

## 五、香港过渡期（1952-1955）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1952-1955 | 旅居香港，为美国新闻处（USIA）写作 | 写了《秧歌》《赤地之恋》，被标上"反共小说"标签 |
| 1953 | 《秧歌》出版（英文版The Rice-Sprout Song） | 在西方出版，但文学评价不如沪港时期作品 |
| 1954 | 《赤地之恋》出版（英文版Naked Earth） | 同上，被认为带有政治立场，削弱了文学纯度 |
| 1955 | 获得美国麦道伟文艺营资助，赴美 | 离开香港，开启美国时期 |

---

## 六、美国早期与赖雅婚姻（1955-1967）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1955 | 抵达美国，入住麦道伟文艺营 | 开始尝试在英文文学圈立足 |
| 1956 | 结识赖雅（Ferdinand Reyher），年长她29岁的美国剧作家 | 她生命中最后一段婚姻的开始 |
| 1956.08 | 与赖雅结婚 | 组成家庭，相互扶持 |
| 1957 | 赖雅第一次中风 | 开始承担照顾病弱丈夫的责任，同时需要写作维生 |
| 1961 | 赴台湾和香港，与宋淇、邝文美重聚 | 维系了最重要的友情纽带 |
| 1963 | 受聘为加州大学伯克利分校中国研究中心研究员 | 开始《红楼梦》版本学研究 |
| 1967.10 | 赖雅去世 | 进入彻底独居状态，此后再未婚嫁 |

---

## 七、《红楼梦》研究与晚期写作（1968-1995）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1968 | 受聘为麻省理工学院（MIT）驻校作家 | 继续学术和创作工作 |
| 1969 | 移居洛杉矶 | 开始漫长的洛杉矶隐居岁月 |
| 1970s | 开始创作《小团圆》，多次修改 | 最私密也最痛苦的创作，涉及胡兰成、原生家庭 |
| 1977 | 出版《红楼梦魇》 | "十年一觉迷考据，赢得红楼梦魇名"，是她最重要的学术成果 |
| 1979 | 《张看》出版 | 晚年散文集，重新整理早年创作 |
| 1983 | 《对照记》出版 | 图文并茂的自传式回顾 |
| 1984 | 《色，戒》收入短篇集 | 以1940年代上海为背景，引发广泛讨论（2007年被李安改编成电影） |
| 1988 | 《续集》出版 | 晚年文字整理 |
| 1990s | 饱受疥虫困扰，频繁搬家，逐渐与外界断绝联系 | 最后几年几乎以洛杉矶公寓为全部世界 |
| 1993 | 告知宋淇，希望死后骨灰撒于太平洋 | 最后的安排 |
| 1995.09.08 | 被房东发现孤独离世于洛杉矶公寓，享年75岁 | 独自完成最后的撤退 |
| 1995.09.30 | 依据遗嘱，骨灰撒于太平洋 | 生于上海，归于大海 |

## 八、身后影响

| 时间 | 事件 | 意义 |
|------|------|------|
| 1995年后 | "张爱玲热"在中国大陆持续升温 | 1980年代"重新发现"后，影响不断扩大 |
| 2007 | 李安改编《色，戒》，获威尼斯电影节金狮奖 | 国际化，引发新一轮张爱玲研究热潮 |
| 2009 | 宋以朗出版《小团圆》 | 引发争议：是否违背张爱玲本意？ |
| 2010 | 《雷峰塔》（The Fall of the Pagoda）出版 | 英文自传体小说，童年视角回顾家族故事 |
| 2012 | 《易经》（The Book of Change）出版 | 英文自传体续集 |
| 2018 | 《张爱玲往来书信集》由宋以朗整理出版 | 大量私人书信首次公开 |

## 九、思想转折点分析

| 转折点 | 时间 | 影响 |
|--------|------|------|
| 父家关禁闭 | 1937年 | 触底才逃离，奠定"战略撤退"的行动模式 |
| 香港战争经历 | 1941-42年 | 战火中见人性底色，《烬余录》《倾城之恋》的历史感来源 |
| 沦陷区成名 | 1943-44年 | 在政治空白地带找到创作出口，但也埋下被边缘化的种子 |
| 与胡兰成离婚 | 1947年 | "因为懂得，所以慈悲"的感情观被现实打破，转向更彻底的内省 |
| 离开上海 | 1952年 | 放弃中文读者主场，接受永久的边缘化与流亡 |
| 赖雅去世 | 1967年 | 进入彻底独居，晚年隐逸生活的真正开始 |
| 洛杉矶隐居 | 1980s-95 | 彻底撤出公众视野，最后的、最彻底的撤退 |
